# Analog Horror: The Doll Cipher

An atmospheric Alternate Reality Game (ARG) styled as analog horror, featuring VHS tape aesthetics, static noise, and floating doll elements. Decode ciphers to uncover the eerie story, with a focus on creative horror rather than intense scares.

## How to Play

1. Open `index.html` in your browser.
2. Click "PLAY TAPE" to start the analog horror experience.
3. Solve each cipher puzzle to advance through the tape.
4. Decode messages using various ciphers: Binary, Caesar, Atbash, Morse, Vigenère, Base64, ROT13, Bacon's, Rail Fence.
5. Experience the unsettling analog horror atmosphere as dolls float around the screen.

## Features

- **Analog Horror Aesthetics**: VHS-style filters, scanlines, static noise, and tape counter.
- **Floating Dolls**: Creepy doll emojis that drift across the screen.
- **Atmospheric Horror**: Subtle glitches and whispers instead of jump scares.
- **Interactive Puzzles**: Decode ciphers to progress through the story.
- **Tape-like Progression**: Each solved cipher advances the "tape" with eerie reveals.

## Hidden Easter Eggs

- Press the Konami code: up, up, down, down, left, right, left, right, B, A to unlock a secret text.
- Click the tape counter 5 times to reveal an on-screen hidden easter message.

## Warning

This ARG contains unsettling themes and analog horror elements. The dolls are always watching...

## Ciphers Included

- **Binary**: Convert binary to text.
- **Caesar Cipher**: Shift letters by 3.
- **Atbash**: Reverse the alphabet.
- **Morse Code**: Dots and dashes.
- **Vigenère**: Keyword-based substitution.
- **Base64**: Decode base64 strings.
- **ROT13**: Rotate by 13.
- **Bacon's Cipher**: A/B binary for letters.
- **Rail Fence**: Transposition cipher.

Each decoded message reveals a piece of the analog horror narrative.
